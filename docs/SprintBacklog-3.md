## BS03: Página e Canal de Contato com Formulários
**Descrição:** Implementar a página de Contato da Build Sync com formulários estruturados para que donos de depósitos de materiais de construção entrem em contato com a equipe comercial.

### Atribuições dos Integrantes:
* **Aleksander (Backlog do Produto):** Mapear os campos obrigatórios do formulário (Nome, E-mail, Telefone, Tamanho do Depósito) e definir o fluxo de sucesso do usuário.
* **Beatriz (Prototipação):** Desenvolver a interface visual da página de contato e o layout amigável do formulário de mensagens.
* **Mateus (Administração do Repositório):** Organizar o recebimento dos arquivos e preparar a estrutura do repositório para futuras validações de scripts.
* **Marcos (HTML5):** Desenvolver o código HTML5 estruturando o formulário com as tags corretas (`<form>`, `<input>`, `<textarea>`, `<button>`).
* **Felipe (CSS):** Estilizar o formulário de contato, aplicando foco nos campos, cores de validação visual e alinhamento profissional na tela.
